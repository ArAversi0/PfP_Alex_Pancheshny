## Оценка COCOMO

Для приближённой оценки использована базовая модель COCOMO. Проект относится к semi-detached/enterprise-like типу: присутствуют web и desktop клиенты, backend, БД, безопасность, тесты и документация.

| Показатель | Значение |
| --- | --- |
| Оценочный размер | 18 KLOC |
| Тип проекта | Semi-detached |
| Формула трудозатрат | `E = 3.0 * KLOC^1.12` |
| Расчёт | `3.0 * 18^1.12` |
| Оценка трудозатрат | примерно 76 человеко-месяцев |
| Формула длительности | `D = 2.5 * E^0.35` |
| Оценка длительности | примерно 11 месяцев для команды |
| Фактический формат | учебный индивидуальный MVP с ограниченным scope |

Вывод: промышленная оценка выше фактической учебной длительности, потому что COCOMO предполагает полный production-цикл, формальную командную разработку, поддержку, внедрение и сопровождение. В курсовом проекте реализован MVP+ объём с акцентом на архитектуру, основные сценарии, тестирование и демонстрационную готовность.
