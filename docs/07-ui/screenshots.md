# Скриншоты для отчёта

## Web

Скриншоты web-части:

1. Главная страница PfP Companion.
2. Страница входа.
3. Страница регистрации.
4. Страница подтверждения email с успешным сообщением.
5. Архив персонажей с несколькими карточками.
6. Лист персонажа в режиме просмотра.
7. Лист персонажа в режиме редактирования.
8. Инвентарь: создание предмета через пустую ячейку.
9. Инвентарь: drag-and-drop и экипированный предмет.
10. Заклинания: карточка заклинания с изображением.
11. Страница лора.
12. Справочник правил.
13. Главная страница админки.
14. Управление пользователями.
15. Персонажи пользователей с группировкой по email.
16. Редактор контента.

## Desktop

Скриншоты desktop-части:

1. Главное меню.
2. Выбор режима персонажей: guest/account.
3. Локальный архив гостевого режима.
4. Архив аккаунта с синхронизированными персонажами.
5. Лист персонажа в режиме просмотра.
6. Лист персонажа в режиме редактирования.
7. Блок характеристик и навыков.
8. Блоки экипировки и состояния.
9. Инвентарь с изображениями предметов.
10. Окно создания предмета.
11. Заклинания и окно создания заклинания.
12. Дополнительная информация.
13. Лор в Extra Content.
14. Справочник правил в Extra Content.
15. Настройки: общие.
16. Настройки: управление.
17. Настройки: музыка и звуки.
18. Запуск packaged `PfP Companion.exe`.
